Minimal workflow

Each branch lives in its own directory so you can switch context without stashing or junk commits.

Why gitx?

  • No more random git stash before switching branch.
  • No more "WIP" commits just to avoid losing local changes.
  • Fast, repeatable context switches across many feature branches.
  • Built directly on top of plain Git and git worktree.
